第二章：CSS 基礎

什麼是 CSS？

CSS（Cascading Style Sheets）是網頁的化妝師，它負責美化 HTML 元素的視覺呈現。 就像給建築物裝潢一樣，CSS 讓我們的網頁變得美觀、專業且易於閱讀。

  • 掌握盒模型的概念和應用
  • 學會基本的佈局技巧
  • 能夠美化中西醫筆記頁面

如何引入 CSS

CSS 的三種引入方式

在靜態網頁中，我們有三種方式來引入 CSS 樣式，每種方式都有其適用場景。

1. 內聯樣式 (Inline Styles)

直接在 HTML 元素中使用 style 屬性

內聯樣式範例 
<h1 style="color: blue; font-size: 24px;">心臟系統</h1>
<p style="background: lightblue; padding: 10px;">心臟是循環系統的核心</p>
優點： 直接、快速、優先級最高
缺點： 難以維護、無法重複使用、混合了結構和樣式

2. 內部樣式表 (Internal Stylesheet)

在 HTML 文件的 <head> 區域使用 <style> 標籤

內部樣式表範例 
<head>
    <style>
        h1 { color: blue; font-size: 24px; }
        .medical-title { background: lightblue; padding: 10px; }
    </style>
</head>
優點： 集中管理、可重複使用、結構清晰
缺點： 僅限單一頁面、檔案可能過大

3. 外部樣式表 (External Stylesheet)

使用 <link> 標籤引入獨立的 CSS 檔案

外部樣式表範例 
<head>
    <link rel="stylesheet" href="css/lesson_2.css">
    <link rel="stylesheet" href="css/common.css">
</head>
優點： 可重複使用、易於維護、支援快取、分離關注點
缺點： 需要額外的 HTTP 請求

💡 最佳實踐建議

  • 優先使用外部樣式表：便於維護和重複使用
  • 內部樣式表用於頁面特定樣式：僅該頁面需要的樣式
  • 內聯樣式僅用於動態樣式：JavaScript 動態修改的樣式
  • 注意載入順序：後載入的樣式會覆蓋先載入的樣式

CSS Reset

為什麼需要 CSS Reset？

不同的瀏覽器對 HTML 元素有預設的樣式，這些預設樣式可能不一致，導致網頁在不同瀏覽器中顯示效果不同。CSS Reset 可以統一這些預設樣式。

基本 CSS Reset 
/* 基本 CSS Reset */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

/* 移除預設樣式 */
h1, h2, h3, h4, h5, h6 {
    font-size: inherit;
    font-weight: inherit;
}

ul, ol {
    list-style: none;
}

a {
    text-decoration: none;
    color: inherit;
}

img {
    max-width: 100%;
    height: auto;
}

🎯 CSS Reset 的好處

  • 統一跨瀏覽器顯示：確保在不同瀏覽器中一致
  • 提供乾淨的起點：從零開始建立樣式
  • 提高開發效率：減少樣式衝突問題
  • 更好的控制權：完全掌控元素樣式

Box-sizing 盒模型控制

什麼是 Box-sizing？

box-sizing 屬性控制元素的寬度和高度計算方式，決定是否包含內距 (padding) 和邊框 (border)。

content-box (預設值)

內容區域 width: 200px

總寬度 = width + padding + border

200px + 20px + 4px = 224px

border-box (推薦)

內容區域 width: 200px

總寬度 = width (包含 padding 和 border)

200px (總寬度固定)
Box-sizing 範例 
/* content-box - 預設值 */
.content-box {
    width: 200px;
    padding: 10px;
    border: 2px solid #ccc;
    box-sizing: content-box;
    /* 實際寬度 = 200px + 20px + 4px = 224px */
}

/* border-box - 推薦使用 */
.border-box {
    width: 200px;
    padding: 10px;
    border: 2px solid #ccc;
    box-sizing: border-box;
    /* 實際寬度 = 200px (固定) */
}

/* 全域設定 border-box */
* {
    box-sizing: border-box;
}

💡 Box-sizing 使用建議

  • 全域設定 border-box：讓所有元素使用統一的盒模型
  • 更直觀的尺寸控制：設定的寬度就是實際寬度
  • 簡化響應式設計：更容易計算和調整佈局
  • 避免意外溢出：減少因 padding 和 border 造成的佈局問題

CSS 選擇器

選擇器的基本概念

CSS 選擇器用來指定要套用樣式的 HTML 元素。就像給不同的人穿不同的衣服一樣。

基本選擇器 
/* 標籤選擇器 */
h1 { color: blue; }

/* 類別選擇器 */
.medical-title { font-size: 24px; }

/* ID 選擇器 */
#heart-info { background: lightblue; }

<div id="heart-info">心臟是循環系統的核心</div>

每個 HTML 元素都可以視為一個盒子，由內到外包含：內容、內距、邊框、外距。

盒模型 CSS 範例 
.medical-card {
    width: 300px;           /* 內容寬度 */
