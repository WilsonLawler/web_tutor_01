第一章：HTML 基礎

什麼是 HTML？

HTML（HyperText Markup Language）是網頁的骨架，它使用標籤（tags）來定義網頁的結構和內容。 就像建築物的鋼筋水泥一樣，HTML 為網頁提供了基本的框架。

🎯 學習目標

  • 理解 HTML 的基本語法結構
  • 掌握常用 HTML 標籤的使用
  • 學會建立語義化的網頁結構
  • 能夠建立基本的中西醫筆記頁面

HTML 語法基礎

標籤的基本結構

HTML 標籤通常成對出現，包含開始標籤和結束標籤：

基本標籤結構 
<標籤名稱>內容</標籤名稱>

實際範例：

<h1>中西醫人體導覽</h1>
<p>這是一個關於人體構造的學習網站</p>

中西醫人體導覽

這是一個關於人體構造的學習網站

HTML 基本結構

🏗️ HTML 文件的基本架構

每個 HTML 文件都必須包含基本的結構標籤，這些標籤定義了網頁的基本框架：

HTML 基本結構 
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>網頁標題</title>
</head>
<body>
    <!-- 網頁內容 -->
</body>
</html>

結構標籤說明：

<!DOCTYPE html>
文件類型聲明

告訴瀏覽器這是 HTML5 文件

<html>
根元素

包含整個網頁內容的根標籤

<head>
頭部區域

包含網頁的元資料和設定

<meta>
元資料標籤

提供網頁的額外資訊

<title>
網頁標題

顯示在瀏覽器標籤頁的標題

<body>
主體內容

包含網頁的可見內容

📋 重要的 Meta 標籤

字符編碼
<meta charset="UTF-8">

指定網頁使用的字符編碼，支援中文顯示

響應式設計
<meta name="viewport" content="width=device-width, initial-scale=1.0">

讓網頁在手機和平板上正確顯示

網頁描述
<meta name="description" content="網頁描述文字">

提供網頁的簡短描述，有助於 SEO

關鍵字
<meta name="keywords" content="關鍵字1, 關鍵字2">

指定網頁的關鍵字，有助於搜尋引擎優化

常用 HTML 標籤

📝 標題標籤 (h1-h6)

標題標籤用於建立網頁的層次結構，從 h1（最重要）到 h6（最不重要）：

標題標籤範例 
<h1>西醫｜臟腑（器官系統總覽）</h1>
<h2>心臟（Cardiac）</h2>
<h3>心臟功能</h3>
<h4>心跳指標</h4>

實際效果：

西醫｜臟腑（器官系統總覽）

心臟（Cardiac）

心臟功能

心跳指標

📄 段落和文字標籤

用於組織文字內容的基本標籤：

段落和文字標籤 
<p>這是一個段落，用於包含文字內容。</p>
<strong>重要文字</strong>
<em>強調文字</em>
<br>換行標籤（自閉合標籤）

實際效果：

這是一個段落，用於包含文字內容。

重要文字 和 強調文字

第一行
第二行（使用 br 標籤換行）

📋 列表標籤

用於建立有序和無序列表：

列表標籤範例 
<!-- 無序列表 -->
<ul>
    <li>心臟（Cardiac）</li>
    <li>肺（Pulmonary）</li>
    <li>肝（Hepatic）</li>
</ul>

<!-- 有序列表 -->
<ol>
    <li>學習 HTML 基礎</li>
    <li>掌握 CSS 樣式</li>
    <li>學習 JavaScript</li>
</ol>

實際效果：

無序列表：
  • 心臟（Cardiac）
  • 肺（Pulmonary）
  • 肝（Hepatic）
有序列表：
  1. 學習 HTML 基礎
  2. 掌握 CSS 樣式
  3. 學習 JavaScript

🔗 連結標籤

用於建立超連結，連接不同的網頁或資源：

連結標籤範例 
<!-- 外部連結 -->
<a href="https://www.example.com">外部網站</a>

<!-- 內部連結 -->
<a href="index.html">返回主頁</a>

<!-- 新視窗開啟 -->
<a href="demo.html" target="_blank">查看示範</a>

實際效果：

外部網站

返回主頁

語義化標籤

🏗️ 語義化標籤的重要性

語義化標籤讓網頁結構更清晰，對搜尋引擎和螢幕閱讀器更友善：

語義化標籤範例 
<header>
    <h1>中西醫人體導覽</h1>
    <nav>
        <ul>
            <li><a href="#chapter1">臟腑系統</a></li>
            <li><a href="#chapter2">骨骼系統</a></li>
        </ul>
    </nav>
</header>

<main>
    <article>
        <h2>心臟功能</h2>
        <p>心臟是循環系統的核心...</p>
    </article>
</main>

<footer>
    <p>© 2025 人體導覽專題</p>
</footer>

語義化標籤說明：

<header>

頁面或區塊的標題區域

<nav>

導航連結區域

<main>

主要內容區域

<article>

獨立的文章內容

<section>

內容的區塊

<footer>

頁面或區塊的結尾

實作練習

🎯 建立中西醫筆記頁面

現在讓我們用學到的 HTML 標籤來建立一個基本的中西醫筆記頁面：

練習任務：

  1. 建立一個基本的 HTML 頁面結構
  2. 使用語義化標籤組織內容
  3. 添加標題、段落和列表
  4. 建立導航連結
完整範例 
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>中西醫人體導覽</title>
</head>
<body>
    <header>
        <h1>中西醫人體導覽</h1>
        <nav>
            <ul>
                <li><a href="#western">西醫觀點</a></li>
                <li><a href="#chinese">中醫觀點</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="western">
            <h2>西醫｜臟腑（器官系統總覽）</h2>
            <article>
                <h3>心臟（Cardiac）</h3>
                <p>心臟是循環系統的核心，負責泵血維持全身循環。</p>
                <h4>重要指標：</h4>
                <ul>
                    <li>心跳速率</li>
                    <li>血壓</li>
                    <li>心電圖</li>
                </ul>
            </article>
        </section>
    </main>

    <footer>
        <p>© 2025 人體導覽專題。僅供教育與參考使用。</p>
    </footer>
</body>
</html>

完整 HTML 架構範本

📋 複製即可使用的基礎 HTML 架構

以下是一個完整的 HTML 架構範本，包含了所有必要的標籤和設定，您可以直接複製使用：

💡 使用說明：

  • 複製下方完整代碼到您的 HTML 檔案中
  • 修改標題、描述和內容部分
  • 根據需要添加或移除區塊
  • 這個架構已經包含了響應式設計和 SEO 優化
完整 HTML 架構範本 
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的網頁標題</title>
    <meta name="description" content="網頁描述，簡短說明網頁內容">
    <meta name="keywords" content="關鍵字1, 關鍵字2, 關鍵字3">
    <meta name="author" content="您的姓名">
    
    <!-- 預載入字體（可選） -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans+TC:wght@300;400;500;700&display=swap" rel="stylesheet">
    
    <!-- 樣式表（可選） -->
    <!-- <link rel="stylesheet" href="css/style.css"> -->
</head>
<body>
    <!-- 頁面標題區域 -->
    <header>
        <h1>我的網站標題</h1>
        <nav>
            <ul>
                <li><a href="#home">首頁</a></li>
                <li><a href="#about">關於</a></li>
                <li><a href="#contact">聯絡</a></li>
            </ul>
        </nav>
    </header>

    <!-- 主要內容區域 -->
    <main>
        <!-- 首頁區塊 -->
        <section id="home">
            <h2>歡迎來到我的網站</h2>
            <p>這是一個使用 HTML 建立的網頁。</p>
        </section>

        <!-- 關於區塊 -->
        <section id="about">
            <h2>關於我們</h2>
            <article>
                <h3>我們的使命</h3>
                <p>提供優質的服務和產品。</p>
                
                <h4>服務項目：</h4>
                <ul>
                    <li>網頁設計</li>
                    <li>程式開發</li>
                    <li>技術諮詢</li>
                </ul>
            </article>
        </section>

        <!-- 聯絡區塊 -->
        <section id="contact">
            <h2>聯絡我們</h2>
            <p>如有任何問題，歡迎與我們聯絡：</p>
            <ul>
                <li>電話：<strong>02-1234-5678</strong></li>
                <li>電子郵件：<em>contact@example.com</em></li>
                <li>地址：台北市信義區信義路五段7號</li>
            </ul>
        </section>
    </main>

    <!-- 頁尾區域 -->
    <footer>
        <p>© 2025 我的網站。保留所有權利。</p>
        <p>最後更新：<time datetime="2025-01-01">2025年1月1日</time></p>
    </footer>

    <!-- JavaScript（可選） -->
    <!-- <script src="js/script.js"></script> -->
</body>
</html>
