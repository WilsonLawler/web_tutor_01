在開始學習 HTML 之前，我們需要了解網頁開發的基本概念、準備必要的開發工具， 以及理解 HTML 如何在瀏覽器中渲染。這堂準備課程將為您建立穩固的基礎。

網頁開發基礎概念

什麼是網頁？

網頁是由 HTML、CSS 和 JavaScript 三種技術構成的文件，透過瀏覽器顯示給使用者。

網頁的三大組成部分：

📄
HTML

網頁的骨架和內容結構

標題、段落、圖片 連結、列表、表格 語義化標籤
🎨
CSS

網頁的樣式和視覺呈現

顏色、字體、間距 佈局、動畫、響應式 視覺效果
⚡
JavaScript

網頁的互動和動態功能

按鈕點擊、表單驗證 動態內容、API 呼叫 用戶互動

瀏覽器如何渲染網頁

從 HTML 到視覺化網頁的過程

當您在瀏覽器中輸入網址或開啟 HTML 檔案時，瀏覽器會執行一系列複雜的處理步驟。

瀏覽器渲染流程：

📥
1. 接收 HTML

瀏覽器從伺服器或本地檔案讀取 HTML 代碼

🔍
2. 解析 HTML

將 HTML 代碼轉換成 DOM（文件物件模型）樹狀結構

🎨
3. 解析 CSS

讀取並解析 CSS 樣式，建立樣式規則

🏗️
4. 建立渲染樹

結合 DOM 和 CSS 建立渲染樹（Render Tree）

📐
5. 佈局計算

計算每個元素的位置和大小（Layout/Reflow）

🖼️
6. 繪製頁面

將元素繪製到螢幕上（Paint）

DOM 樹狀結構範例：

html
head
title 我的網頁
body
h1 歡迎來到我的網站
p 這是一個段落

VS Code 安裝與設定

Visual Studio Code 安裝步驟

VS Code 是目前最受歡迎的免費程式碼編輯器，擁有豐富的擴展生態系統。

1

下載 VS Code

前往官方網站下載適合您作業系統的版本

🌐 前往 VS Code 官網下載
2

安裝 VS Code

執行下載的安裝檔，按照指示完成安裝

💡 Windows 安裝提示：
  • 建議勾選「Add to PATH」選項
  • 選擇「Open with Code」檔案關聯
  • 勾選「Create a desktop icon」建立桌面圖示
  • 勾選「Add 'Open with Code' action to Windows Explorer」
  • 安裝完成後重新啟動電腦
🪟 Windows 特定設定：
📁
檔案總管整合

安裝後可在檔案總管中右鍵選擇「Open with Code」

⌨️
快捷鍵設定

Windows 快捷鍵：Ctrl+Shift+P（命令面板）

🔧
終端機整合

使用內建終端機：Ctrl+`（反引號）

3

安裝 Live Server 擴展

Live Server 是網頁開發的必備擴展，提供即時預覽功能

🔍 點擊左側「擴展」圖示
📝 搜尋「Live Server」
⬇️ 點擊「安裝」按鈕
✅ 安裝完成後重新載入 VS Code
4

建立第一個專案

建立資料夾並開始您的第一個網頁專案

📁 建立專案資料夾

在桌面或您喜歡的位置建立一個新資料夾，命名為「my-first-website」

🪟 Windows 操作步驟：
  1. 在桌面空白處按右鍵
  2. 選擇「新增」→「資料夾」
  3. 將資料夾命名為「my-first-website」
  4. 按 Enter 確認
📂 在 VS Code 中開啟

使用 VS Code 開啟這個資料夾

🪟 Windows 開啟方法：
方法一：在資料夾上右鍵 → 選擇「Open with Code」
方法二：開啟 VS Code → File → Open Folder → 選擇資料夾
方法三：拖拽資料夾到 VS Code 視窗中
📄 建立 HTML 檔案

在資料夾中建立一個名為「index.html」的檔案

🪟 Windows 建立檔案：
  1. 在 VS Code 左側檔案總管中，右鍵點擊資料夾
  2. 選擇「新增檔案」
  3. 輸入檔案名稱「index.html」
  4. 按 Enter 確認

Live Server 使用指南

如何使用 Live Server 進行開發

Live Server 讓您能夠即時預覽網頁變更，大幅提升開發效率。

Live Server 工作流程：

📝
1. 編寫代碼

在 VS Code 中編寫或修改 HTML 代碼

→
💾
2. 儲存檔案

使用 Ctrl+S 儲存檔案

Ctrl+S 儲存檔案
→
🔄
3. 自動重新載入

瀏覽器自動重新載入，顯示最新變更

Live Server 主要功能：

⚡
即時重新載入

修改代碼後自動更新瀏覽器，無需手動重新整理

🌐
本地伺服器

建立本地 HTTP 伺服器，模擬真實網頁環境

📱
多裝置預覽

支援在不同裝置上預覽響應式設計

🔧
自訂設定

可調整端口號、延遲時間等設定

啟動 Live Server 的方法：

方法一：右鍵選單
  1. 在 HTML 檔案上按右鍵
  2. 選擇「Open with Live Server」
  3. 瀏覽器會自動開啟並顯示網頁
方法二：狀態列
  1. 開啟 HTML 檔案
  2. 點擊 VS Code 底部狀態列的「Go Live」
  3. 選擇要開啟的檔案
方法三：快捷鍵
  1. 開啟 HTML 檔案
  2. 使用快捷鍵 Alt+L Alt+O
  3. 或使用命令面板 (Ctrl+Shift+P)
🪟 Windows 常用快捷鍵：
Ctrl+Shift+P 命令面板
Ctrl+` 開啟終端機
Ctrl+S 儲存檔案
Ctrl+Z 復原
Ctrl+Y 重做
Ctrl+F 搜尋

實作練習

🎯 建立您的第一個網頁

現在讓我們實際操作，建立您的第一個網頁並使用 Live Server 預覽。

練習步驟：

1
建立專案資料夾

在桌面建立名為「my-first-website」的資料夾

2
在 VS Code 中開啟

使用 VS Code 開啟這個資料夾

3
建立 HTML 檔案

建立「index.html」檔案並輸入以下代碼：

index.html 
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一個網頁</title>
</head>
<body>
    <h1>歡迎來到我的網站！</h1>
    <p>這是我使用 HTML 建立的第一個網頁。</p>
    <p>我學會了如何使用 VS Code 和 Live Server 進行開發。</p>
</body>
</html>
4
使用 Live Server 預覽

右鍵點擊 HTML 檔案，選擇「Open with Live Server」

5
測試即時更新

修改 HTML 內容並儲存，觀察瀏覽器自動更新
